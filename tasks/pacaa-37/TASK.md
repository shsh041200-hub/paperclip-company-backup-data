## 분류

이 작업은 **사이드 프로젝트**다. Vision 또는 Goal 트리 직접 기여
아님. SG-3 임계 경로 외부 작업. 보드 편의성 향상 목적.

향후 보드가 던지는 비슷한 요청도 동일 무게로 처리할 것:

* 단일 티켓 (분해 최소화)
* 결재 게이트 최소화
* 본 작업 후순위
* 가벼운 보고

## 목적

* 데스크톱: 회사 상태(Goal 트리 진척률) 시각화
* 모바일: 결재 필요 항목을 외부 알림으로 즉시 수신

## 우선순위

현재 Live 작업 후순위. PACAA-34, PACAA-30 시리즈, Pair 1 채용
안정화 후 시작. SG-3 임계 경로 영향 금지.

## 요구사항

### 1. 대시보드 (데스크톱용)

* Goal 트리 진척률: Vision → TG → SG → P

업데이트 주기: 10분 이내.
도구:  대안 분석 후 보드 추천.

### 2. 보드 액션 큐 알림 (모바일용)

내가 결재 / 승인 / 검토해야 할 항목이 발생 시 핸드폰으로 알림 도달.
방식 무관 — 가장 안정적이고 무료인 채널 선택.

24/7 가능해야함 매일매일 하루종일 가능해야함

## 비용 제약 (엄격)

* 영구 무료 서비스만 사용
* "첫 달 무료", "30일 체험", "X 사용량까지 무료 후 유료" 같은
  조건부/한시적 무료는 유료로 간주하고 제외
* 현재 무료지만 향후 유료 전환 위험이 있는 서비스는 명시 후 보드 결정

## 기타 제약

* 1회 셋업 후 유지보수 최소화
* 한국 안정성 우선

## 산출물

CTO 위임. 단일 티켓. 1주 내 완성.
도구 선택 시점에만 보드 결정 게이트 1회.

완료 시 보드 사인오프 요청.
