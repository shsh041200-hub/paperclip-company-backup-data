## 배경

[PACAA-322](/PACAA/issues/PACAA-322) — 비교 페이지 하단 견적 의뢰 폼 신설 계획. buyer 이메일·회사명·수량·마감일·요구사항을 수집 후 선택된 vendor 3곳에 이메일로 전달.

## 자문 요청 사항

1. **PIPA 수집 동의 문구**: 이메일·회사명·요구사항 수집 시 필요한 최소 동의 문구 초안 (개인정보보호법 제15조/제17조 기준).
2. **보관 기간**: lead 데이터 적정 보관 기간 — 거래 성사 여부와 무관하게 단순 lead 인 경우.
3. **처리 방침 update 필요성**: 기존 처리 방침에 'lead capture' 항목 추가 필요 여부.
4. **통신판매업 신고 영향**: 견적 의뢰 폼이 전자상거래법상 '통신판매중개' 또는 '통신판매업' 신고 의무를 유발하는지.
5. **제3자 제공 동의**: vendor 에 buyer 정보 전달 시 별도 제3자 제공 동의 필요 여부.
