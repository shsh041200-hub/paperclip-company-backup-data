Write the runbook the CEO follows on every new hire so that adding an 11th agent takes <1 heartbeat.

## Scope

- Step-by-step procedure: hire request → board approval → agent creation → instructions install → skill sync → first-heartbeat smoke test
- Templates and pointers wired in (PACAA-7.1, 7.2, 7.3)
- Failure-mode checklist (what to verify after first heartbeat)
- Decision tree for model assignment (Sonnet 4.6 default, Opus only for CEO)

## Definition of done

- Onboarding runbook committed (markdown) and linked from PACAA-7
- Validated by re-hiring the first agent (CTO) following the runbook end-to-end
- Iterated based on CTO re-hire findings before bulk re-hire
