## 작업 범위

다음 4가지를 보드에 제출:

1. **한국 패키징 시장 카테고리 통상 분류**
   - 산업 표준(KSIC, 통계청 사업자 분류) 기반 분류 체계
   - 산업협회/업계 통상 분류 (예: 한국포장협회, 한국포장기술사회)
   - 기존 디렉토리(엠파크, 위키트 등)의 카테고리 분류 패턴

2. **8개 후보 카테고리 명시**
   - 각 카테고리에 대해:
     - 한국 사업체 수 추정 (통계청 KOSIS + 업종코드)
     - 월간 검색량 (Naver Keyword Planner + Google Keyword Planner)
     - buyer-intent 강도 (sourcing/quote/comparison 트래픽 점유)
     - 현재 Packlinx 등록 vendor 수 (1,396개 중 매핑 가능 분량)

3. **동시 vs 순차 진행 권장**
   - 8개 카테고리를 동시에 SG-1 90% 등록 추진할 것인지
   - 또는 우선 4개 → 후속 4개 sequential 추진할 것인지
   - 권장과 근거

4. **4개로 줄일 경우의 우선순위**
   - SG-1 Kill criterion 트리거 시 (6주차 분모 미완 또는 평균 < 50%)
     보존할 4개 카테고리
   - 우선순위 순서와 근거
