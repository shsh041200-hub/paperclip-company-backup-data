Write specialized skills per role so each agent has structured tools beyond the common skill set.

## Scope (one skill per role at minimum)

- CTO — architecture decision template, code-review checklist for Next.js 16/Supabase
- CMO — content brief template, Korean B2B SEO playbook
- COO — vendor onboarding SOP, takedown procedure pointer
- HoP (Head of Product) — ticket grooming template, success-metric framework
- Frontend / Backend engineer — language-specific runbook scoped to the actual stack
- Designer — DESIGN.md update procedure, brand-asset handoff format
- HeadOfSales — outbound script library, pricing-negotiation checklist
- LegalCounsel — Korean compliance checklist, takedown drafting template

## Definition of done

- One SKILL.md per role committed under company skill registry
- Each new agent's `desiredSkills` includes its role-specialized skill
