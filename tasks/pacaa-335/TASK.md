## 배경

GSC 3개월 데이터 분석 결과:
- `/guides/shipping-box-pricing`: 0클릭, 8노출, **평균 순위 4.0**
- 순위 4위인데 CTR 0% → 스니펫이 클릭 유도 실패
- 쿼리별 데이터 비공개 (개별 쿼리 노출수가 GSC 임계치 미달)

## 문제 진단

**현재 제목** (63자):
`배송 박스 가격 완전 가이드 — 사이즈별·수량별 단가 비교 | Packlinx`
- "완전 가이드" 문구 — 정보성 느낌, 가격 확인 의도 검색자에게 약함
- 연도 없음
- 실제 가격 숫자 미노출

**현재 메타 설명** (55자, 매우 짧음):
`택배 박스 사이즈별·수량별 단가, 가격 결정 요인, 업체 비교 기준을 정리합니다.`
- 수동적 어조 ("정리합니다")
- 구체적 숫자 없음
- CTA 없음
- 검색자가 클릭해야 할 이유 없음

## 권장 변경사항

**새 제목:**
`택배 박스 가격표 (2026) — 사이즈별 단가·수량 할인·업체 비교 | Packlinx`
- "가격표" 키워드 직접 포함
- 연도 추가
- "완전 가이드" 제거 → 실용성 강조

**새 메타 설명** (~95자):
`택배 박스 1호 130~180원, 2호 160~220원 (1,000개 기준). 수량 5,000개↑ 시 30~40% 할인. Packlinx에서 무료 견적 비교.`
- 실제 가격 숫자 전면 배치
- 수량 할인율 명시
- 무료 견적 CTA

## 예상 효과

순위 4위 + 개선된 스니펫 → CTR 5~8% 달성 시 3개월 기준 8노출 × 6% ≈ 0.5클릭
(현재 대비 유의미한 개선, 노출수 증가 시 더 큰 영향)
