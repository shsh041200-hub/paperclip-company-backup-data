PACAA-716 soak 분석 결과 도출된 즉각 조치 항목:

1. **Sitemap 포함 확인 (CTO)** — GSC "감지된 참조 사이트맵 없음". /guides/flexible-packaging-guide가 sitemap.xml에 포함되어 있는지 CTO 확인 및 미포함 시 추가.

2. **내부 링크 추가** — GSC 참조 페이지 0건. 연포장재 카테고리 페이지 또는 관련 랜딩 페이지에서 가이드로 인바운드 링크 최소 3개 추가.

3. **골판지 쿼리 cross-ranking 해소** — ab골, a골, e골 등 골판지 flute 쿼리에서 연포장재 가이드가 ranking 중. 페이지 내 골판지 언급 최소화 또는 주제 명확화.

4. **메타 description CTR 개선** — 연포장재 핵심 쿼리 CTR 0%. 클릭 유도 문구 강화.

5. **GSC 색인 생성 요청** — URL 검사 도구에서 색인 생성 요청 실행.

**4주 목표 (2026-06-12):** 연포장재 쿼리 노출 50+, CTR 3%+, 골판지 cross-ranking 0
