## PACAA-240 트리거 Surface

- **Surface 1** — 사이트 외부 노출 콘텐츠 신규 발행 (가이드 5개 슬롯: TL;DR·Callout·Checklist·FAQ·Sidebar·end-CTA)
- **Surface 3** — SEO 메타·내부링크·콘텐츠에 규제 기관명(식약처·KC·CE·FCC) 및 법령명 포함 가능성

## 자문 요청 대상 가이드 (5개)

1. `glass-metal-container-guide` — 유리·금속 용기 가이드 (소재)
   - 리스크: 일반론 위주, 리스크 낮음 예상. BPA-free 내면 코팅 표현 확인 요청.

2. `food-packaging-materials` — 식품 패키징 소재 (산업별·식품)
   - 리스크: 식품위생법 §9 + 식약처 「기구·용기·포장의 기준 및 규격」 — 이행성 시험 성적서 표현 적합성 확인 요청
   - 콘텐츠 내 "이행성 시험 성적서 수령 권장" 표현이 법적 의무인 것처럼 오해될 여지 없는지 확인

3. `cosmetic-packaging-box` — 화장품 패키징 박스 (산업별·화장품)
   - 리스크: 화장품법 §10 표시 의무 (전성분·사용기한·사용방법 필수 기재) — 표현 적합성 확인 요청
   - 재활용 표기 어조: "처분이 가능합니다" 표현 사용 여부 확인
   - 과대 포장 기준 (2차 포장 15% 이내) 표현의 정확성 확인

4. `electronics-packaging-design` — 전자제품 패키징 디자인 (산업별)
   - 리스크: KC·CE·FCC 안전 인증 표현 — "인증 마크 보유 여부 확인" 톤으로 표현, 단정적 "인증이 필요합니다" 표현 회피
   - ISTA·ASTM 기준 언급의 적합성 확인

5. `packaging-accessories-guide` — 포장 부자재 가이드 (산업별 또는 소재)
   - 리스크: 일반론 위주, 리스크 낮음 예상.
