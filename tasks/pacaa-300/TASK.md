회사 모든 에이전트 md(ceo제외)/skill 파일 영문화 작업.

## 배경

보드의 수동 작업으로 ceo 에이전트의 HEARTBEAT/SOUL/AGENTS md 파일들을 압축 후 영문화 효과 검증됨. LLM 처리 부담
감소. 다른 에이전트 md(ceo제외 이미함)/skill 파일도 동일 원칙 적용.

## 범위

대상 파일:

* 에이전트 (CTO/Backend/CMO/Legal/Frontend) 의
  HEARTBEAT.md, SOUL.md, AGENTS.md
* 모든 SKILL.md 파일

## 변환 룰

### 변환 대상 (영문화)

* 시스템 룰 / 절차 / 지시문
* Phase 설명 / 체크리스트
* 보강 표시 / PACAA 참조 본문
* 일반 한국어 설명문

### 변환 제외 (한국어 보존)

* 한국 회사명 / 브랜드명 (이음, 위키트, 다나와, 잡코리아,
  쿠팡, 네이버 등) — 고유명사
* 한국어 사례 / 인용문 ("3일 안에 X" 같은 anti-pattern 예시)
* 보드 보고용 한국어 출력 형식 (텔레그램 알림 형식 등)
* Regex 패턴의 한국어 검색 문자열 ("에스컬레이션 → CEO" 등)

## 보존 의무

* 모든 룰 의미 100% 보존
* 모든 PACAA 참조 보존
* 모든 보강 (R1\~R8 등) 보존
* 마크다운 구조 동일
* 시스템 지장 0

## 작업 절차

1. 작업 전 모든 대상 파일 백업
   `cp {file}.md {file}.md.backup-2026-05-07`
2. 파일별 영문화 변환
3. 검증:
   * 각 파일 변환 전후 룰 개수 일치
   * 한국 회사명 / regex 패턴 보존 확인
   * 마크다운 구조 동일 확인
4. 변환 결과 보드에 보고:
   * 변환된 파일 목록
   * 각 파일 토큰 감소량
   * 보존 검증 결과

## 검증 후 다음 단계

영문화 후 하트비트를 실행해 1\~2 wake raw 검증:

* 모든 에이전트가 영문 파일 정상 처리하는가
* 한국 회사명 / 보드 보고 형식 보존됐는가
* 시스템 작동 정상인가

미작동 발견 시 백업에서 롤백.

## 우선순위

다른 결정 영향 금지.
