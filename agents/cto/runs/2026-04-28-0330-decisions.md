**Ticket:** PACAA-19 — eco/fresh boolean facet filters + sitemap update
**Goal ancestry:** PACAA-15 (업체 카테고리 개선안) → user discovery improvement

### Options considered

1. **Add boolean toggle buttons inline (chosen)** — Add `?eco=true` / `?fresh=true` as simple toggle buttons alongside existing dropdown filters. Minimal UI change, consistent with existing `sample` toggle pattern.
2. **Add eco/fresh as dropdown filters** — Create new dropdown sections. Overkill for boolean values; dropdowns are suited for multi-value facets.

### Benchmark
Naver SmartStore and Coupang Business both use simple boolean toggles for attribute filters (free shipping, eco-certified). Toggle pattern is proven for binary facets in Korean B2B/B2C contexts.

### Decision + reasoning
Option 1. Boolean toggles match the existing `샘플 가능` pattern already in the filter bar. No new component abstraction needed. Color-coded (green for eco, sky for fresh) for quick visual distinction.

### Reversibility
Two-way door. All changes are additive: new DB column, new URL params, new UI toggles. No schema removal or data mutation.
